# SHRI-Factorial

Question #13 in [SHRI 2013] (http://company.yandex.ru/job/vacancies/shri_2013.xml)

Write the function on JavaScript, which show list of numbers, which equal sum of their own factorial.

Example:
4! + 0! + 5! + 8! + 5! = 40585

## Result
```sum_factorial_list(1000000);

[ 40585, 145, 2, 1, 0 ]```
